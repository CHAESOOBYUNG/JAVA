멤버변수(Member Variable)
• 클래스 안에서 선언 된 변수
• 객체를 생성할 때 각 데이터 타입의 기본 값으로 자동 초기화
• 기본 값 : 정수형 0, 실수형 0.0, 논리형 false, 객체 null

지역변수(Local Variable)
• 메서드 안에서 선언된 변수
• 사용하기 전에 반드시 초기화 해야 함

생성자(Constructor)

- 반환 타입을 적지 않고 클래스 이름과 대/소문자까지
똑같이 만들어야 합니다.
- 생성자를 만들지 않을 경우 자바 가상
머신(JVM: Java Virtual Machine)이 대신해서 기본 생성자를 자동으로 하나 만들어 줍니다.
- 생성자의 이름은 클래스 이름과 대/소문자까지 정확히 같아야 합니다. 
- 반환 유형을 적지 않습니다.
- 생성자는 다른 유형으로 여러개 생성 가능

서 메모리를 해제하고 수거.
